ID
ブログの管理番号です。
ステータス
ブログの公開状態を変更します。
名前
ブログの名前を変更します。(日本語可)
ドメイン
ブログのドメイン名を変更します。
コードネーム
URL上で表示されるカテゴリーの名前(ディレクトリ名)を変更します。
インデキシング
チェックされたブログはブログ一覧に表示されなくなります。しかし「非表示」ではないのでURLを入力するとブログが表示されます。
トップページのdescription
ブログトップページ専用のメタタグの「説明文」を入力してください。検索サービスの結果や、Facebookでページを紹介されたときの内容に影響します。
トップページのkeywords
ブログトップページ専用のメタタグの「キーワード」を半角カンマ区切りで入力してください。検索サービスの結果に影響します。
日付＋タイトル新着の見出し
トップページとサイドメニューに使用されている、日付とタイトルが表示されるエントリー新着の「見出し」を入力してください。
カテゴリーを絞る場合はカテゴリーの表示名、特に絞らない場合は「更新情報」が無難です。
写真＋概要＋タイトル新着の見出し
トップページに使用されている、写真と概要とタイトルが表示されるエントリー新着の「見出し」を入力してください。
カテゴリーを絞る場合はカテゴリーの表示名、特に絞らない場合は「新着情報」が無難です。
Facebookコメント
Facebookコメントを有効にする場合はチェックを入れてください。有効にした場合は、「管理者のFacebookID」の入力が必要です。
使用するSNSボタン
トップページ・エントリーページに表示する、ソーシャルメディアボタンを選択してください。
TwitterのID
連携させたいTwitterの登録IDを入力すると、Twitterへのリンクが表示されます。
mixiの開発者キー
トップページ・エントリーページにmixiイイネ！ボタンを表示させたい場合は、mixiの開発者キーを入力してください。
FacebookページのURL
連携させたいFacebookページのURLを入力すると、Like Boxが表示されます。
管理者のFacebookID
管理者のFacebookIDを入力すると、Facebookコメントを管理したり、サイトをFacebookのアクセス解析（ドメインインサイト）に登録できるようになります。
og:image
Facebookやmixiで紹介されたときに自動表示する画像をアップロードしてください。ブログのスクリーンショット、もしくは会社のロゴが無難です。
アクセス解析サービスのコード
Google Analyticsなどの、アクセス解析のコードを貼り付けてください。表示位置は<head>要素の終了直前となります。
カラー
ブログの配色を選択してください。未設定の場合はグレーになります。
ヘッダのロゴ画像
ヘッダに表示するロゴ画像をアップロードしてください。右にブログタイトルが表示されるので、正方形に近い画像が望ましいです。
トップページのメイン画像
メイン画像をアップロードしてください。複数枚設定すると、上から順に表示するスライドショー形式となります。各画像をクリックしたときのリンク先も指定できます。
