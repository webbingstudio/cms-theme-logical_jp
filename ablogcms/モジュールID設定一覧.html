logical_jp_acms :: モジュールID設定一覧

logical_jpのa-blog cms版の導入時に、コンフィグを含むフルインストールができない場合は、モジュールIDを手入力で作成する必要があります。

管理ページのメニューから「カスタマイズ管理＞モジュールID」を選択し、以下の通りIDと設定を定義してください。

  • 任意 アイコンがある設定は、任意変更が可能です。
  • 差分のみ記載しています。モジュールID詳細の「グローバル設定」「階層：ブログ」は、基本的に変更しません（下階層での利用を許可しない）。
  • サブメニュー：バナー（subBanner）
  • サイトマップ（sitemap）
  • サブメニュー：アーカイブ（subArchive）
  • ホーム：日付＋タイトル新着（homeHeadline）
  • サブメニュー：新着（subRecentList）
  • ローカルナビ（localNav）
  • インデックス：写真＋タイトル＋概要新着（indexSummary）
  • ホーム：ピックアップ（homeSummaryPickup）
  • ホーム：写真＋タイトル＋概要新着（homeSummary）
  • エントリー：関連記事（entryRelational）
  • グローバルナビ（globalNav）
  • フッタナビ（footerNav）
  • ヘッダナビ（headerNav）

適用モジュール Category_EntryList

名前 サイトマップ
説明 サイトマップを表示します。

モジュール設定

カテゴリー

インデキシング チェックを入れる
階層 1
エントリー

表示数 5　任意

エントリーの有無 チェックを入れる　任意

モジュールID詳細

適用モジュール Entry_ArchiveList
id subArchive
名前 サブメニュー：アーカイブ
説明 サブメニュー：月別アーカイブリスト ※「カテゴリーID」欄でIDを指定すると、そのカテゴリー以下に属するエントリーのみが表示されます
